# Parthenon

The official website for the [AthenaUS](https://discord.com/invite/5dzECDz) Discord community.

## Features

- **Games** — Play Wordle and Blackjack to earn in-game currency
- **Dashboard** — Track your stats and total in-game currency
- **Discord & Twitch integration** — Link your accounts and manage your profile

## Tech Stack

- [Next.js](https://nextjs.org/) 16 / React 19 with TypeScript
- [Clerk](https://clerk.com/) for authentication
- [MongoDB](https://www.mongodb.com/) via Mongoose
